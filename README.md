# Portfolio

Next.js App Router portfolio with MDX-powered projects and blog, Tailwind styling, and shadcn/ui primitives.

## Scripts
- `npm run dev` — start dev server
- `npm run build` — production build
- `npm run start` — run built app
- `npm run lint` — lint
- `npm run format` — format with Prettier

## Structure
See `app/`, `components/`, `content/`, `config/`, and `lib/` for routes, UI, content, metadata, and utilities.

## Environment
- `NEXT_PUBLIC_SITE_URL` — base URL for metadata

Replace `public/og.png` with a real 1200x630 image.
